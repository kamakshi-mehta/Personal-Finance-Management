# PFM: Personal Finance Management Application (MERN + FastAPI)

This is a personal finance management boilerplate project that links a React frontend, Node/Express backend, and Python FastAPI AI microservice.

## Ports
- **Frontend**: http://localhost:3000 (React + Vite)
- **Backend**: http://localhost:5000 (Express)
- **AI Microservice**: http://localhost:8000 (FastAPI)

## Getting Started
1. Run `npm install` at the root.
2. Run `npm run setup` to install all frontend, backend, and Python virtual environment dependencies.
3. Run `npm start` to run all services concurrently in a single terminal.
